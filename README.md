# Salary Predictor — Data Science Job Market

A machine learning web app that predicts data science salaries based on 
experience, location, and skills. Built as part of my MSc Statistics & 
Data Science portfolio to explore what drives earning potential in the 
UK and global data science job market.

## What it does
- Predicts salary range based on user profile inputs
- Compares Random Forest vs XGBoost model performance
- Shows which features (experience, location, skills) matter most
- Deployed as an interactive Streamlit app

## Why I built this
As an MSc student entering the job market, I wanted to understand what 
factors actually drive salary — not just anecdotally, but statistically. 
This project let me combine regression modelling with a real question 
I had.

## Tech Stack
- Python, Pandas, Scikit-learn, XGBoost
- SHAP for feature importance
- Streamlit for deployment
- Kaggle Data Science Jobs dataset
